# TUMELO UTLWANANG BOJOSI LIVE FINACIAL PORTFOLIO

This project is set up as a hosted-web-portal-ready static site.

## What it includes

- Personal income and expense tracking
- Bank balance tracking for 3 accounts
- Masked card profile tracking
- Personal debt tracking with interest
- Micro-lending investment tracking
- Exportable financial statement
- Collapsible dashboard panels
- BWP to USD asset conversion

## Hosting options

### Netlify

1. Create a new site from this repository.
2. Use the project root as the publish directory.
3. No build command is required.

### Vercel

1. Import this repository as a static project.
2. No framework preset is required.
3. The included `vercel.json` handles basic headers.

### GitHub Pages

1. Push these files to the repository.
2. Open repository `Settings`.
3. Open `Pages`.
4. Set source to deploy from branch `main`.
5. Save.

## Important note

This version is hosting-ready, but it is still a frontend-only portal. Real user accounts, passkeys, biometrics, true 2FA, and secure bank integrations require a backend and HTTPS hosting with authentication services.
